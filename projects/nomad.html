Nomad
Github Repository

With Nomad, I've attempted to create a locally run Python application that visualizes your journeys across the world. The application extracts EXIF GPS data from images that you upload in order to plot them on a map, reconstruct the routes you've taken, and calculate metrics. calculate metrics.

Features:

  • A rendering engine which:
    • Plots images on a map
    • Draws polylines using OSMnx network graphs to approximate routes traveled
    • Iteratively measures distance between coordinates
  • Interactive Folium map
  • Dashboard that displays buttons, trips, and metrics
  • Thumbnail generation via Pillow to show image previews
  • Cache that encodes graph data in binary for efficient plotting


This project has been heavily inspired by my experience reading The Innocents Abroad by Mark Twain. The humorous, often abstract writings about his adventures around Europe and the Middle East reminded me of how, for a large portion of history, travel was only about self-fulfillment.
There was no real need to have our images, locations, and thoughts stored in some far off cloud service, where they are treated as sellable data and nothing more. That was pretty cool.
